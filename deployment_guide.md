# دليل نشر تطبيق المنصة الإحصائية (Deployment Guide)

لنشر موقعك على الإنترنت وجعله متاحاً للجميع، أنصحك باستخدام منصة **Render** لأنها تدعم تشغيل تطبيقات Python/Flask بسهولة وتوفر خطة مجانية جيدة.

## الخطوة 1: تجهيز الملفات (تمت)
لقد قمتُ بالفعل بإضافة الملفات الضرورية للمشروع:
1.  **`requirements.txt`**: يحتوي على المكتبات المطلوبة (أضيفت إليه `gunicorn` للخادم).
2.  **`Procfile`**: ملف يخبر المنصة كيفية تشغيل التطبيق.

## الخطوة 2: رفع الكود على GitHub
لنشر الموقع، يجب أولاً رفع الكود على موقع GitHub.
1.  أنشئ حساباً على [GitHub.com](https://github.com).
2.  قم بإنشاء مستودع جديد (New Repository) وسمّه مثلاً `stat-platform`.
3.  ارفع ملفات المشروع (`app.py`, `analysis.py`, `requirements.txt`, `Procfile`, ومجلدي `static` و `templates`) إلى هذا المستودع.

## الخطوة 3: النشر على Render
1.  اذهب إلى [Render.com](https://render.com) وأنشئ حساباً.
2.  اضغط على **New +** واختر **Web Service**.
3.  اختر **Build and deploy from a Git repository**.
4.  اربط حساب GitHub الخاص بك واختر المستودع `stat-platform` الذي أنشأته.
5.  في صفحة الإعدادات، تأكد من البيانات التالية:
    *   **Runtime**: Python 3
    *   **Build Command**: `pip install -r requirements.txt`
    *   **Start Command**: `gunicorn app:app` (هذا مهم جداً)
6.  اختر الخطة المجانية (Free) واضغط **Create Web Service**.

## الخطوة 4: الانتظار والتشغيل
*   ستبدأ المنصة في بناء التطبيق (قد يستغرق 3-5 دقائق).
*   عند الانتهاء، ستعطيك رابطاً خاصاً بموقعك (مثل `https://stat-platform.onrender.com`).
*   شارك هذا الرابط مع من تريد!

---

**ملاحظة هامة:**
بما أن التطبيق يقوم بحسابات إحصائية ثقيلة (مثل SEM/EFA)، الخطة المجانية قد تكون بطيئة قليلاً في البداية (Spin down if inactive)، لكنها كافية للتجربة والاستخدام العادي. إذا أردت أداءً أسرع، يمكنك الترقية لخطط مدفوعة لاحقاً.
